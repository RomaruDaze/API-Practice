Introduction

DeadlyDeathDeadliest is a indie rouge-like games. This game's goals is to challange players to beat the games as fast as possible while having the highest score. You need to fight your way through to defeat Demon God. But to be able to reach him, you need to defeat all of his god-like retainers

DeadlyDeathDeadliest uses API to manage players' and enemies' status. The API is also used to manage the game's savedata. If you are interested in developing games, contact me!
